Features:

-  You can set a minimum quantity and a multiple quantity on the
   product.
-  if you try to sell less than minimum the number will be set to the
   minimum quantity.
-  if you set multiple quantity you can sell only in multiple of this
   number.
-  if set to 0 neither minimum nor multiple quantity will be checked.
